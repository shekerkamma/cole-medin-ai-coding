# Technologies

Technology and repository notes linked from the Hyundai use cases.

* [ByteTrack](bytetrack.md) - High-performance multi-object tracker — associates detections across video frames for person and vehicle tracking.
* [MediaPipe](mediapipe.md) - Google's lightweight ML framework for on-device pose, hand, and face tracking. Runs in real time on edge devices without GPU.
* [Neo4j](neo4j.md) - Graph database for component-to-vehicle genealogy tracking — instant traversal queries for recall investigation and root-cause analysis.
* [PaddleOCR](paddleocr.md) - Industrial-grade OCR by PaddlePaddle — text detection + recognition for VIN plates, part labels, and serial numbers in challenging industrial lighting.
* [SlowFast](slowfast.md) - Facebook Research's SlowFast networks for video understanding — dual-pathway architecture that captures both spatial and temporal features.
* [XGBoost](xgboost.md) - Gradient boosting library — fast, accurate ML for tabular sensor data. The go-to model for predictive quality on structured production data.
* [anomalib](anomalib.md) - Industrial anomaly detection library by OpenVINO. Implements PatchCore, FastFlow, PaDiM, and other algorithms that train on normal-only images — no labeled defect data needed.
* [jetson-inference](jetson-inference.md) - NVIDIA Jetson edge deployment toolkit — TensorRT optimized inference for computer vision models on Jetson Orin / Nano.
* [librosa](librosa.md) - Audio analysis library — mel spectrogram extraction, MFCC, chroma features for acoustic fault detection.
* [lifelines](lifelines.md) - Survival analysis library — Weibull, Cox proportional hazards, and Kaplan-Meier models for time-to-failure estimation.
* [mmdetection](mmdetection.md) - OpenMMLab detection framework — ResNet, EfficientNet, DETR, Mask R-CNN, and 200+ model configs for object detection and instance segmentation.
* [mmpose](mmpose.md) - OpenMMLab pose estimation — HRNet, ViTPose, and other architectures for human and object keypoint detection.
* [mmpretrain](mmpretrain.md) - OpenMMLab pre-training framework — fine-tune image classifiers (ResNet, EfficientNet, ViT) on custom defect categories.
* [pypylon](pypylon.md) - Python wrapper for Basler industrial camera SDK (pylon). Captures frames from GigE and USB3 Basler cameras for inspection pipelines.
* [supervision](supervision.md) - Roboflow's computer vision toolkit — polygon zone detection, tracking overlays, and annotation utilities.
* [ultralytics](ultralytics.md) - YOLOv8/YOLOv9 — real-time object detection, segmentation, pose estimation, and classification. The workhorse model across 4 use cases.
