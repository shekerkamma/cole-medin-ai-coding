# pypylon

Python wrapper for Basler industrial camera SDK (pylon). Captures frames from GigE and USB3 Basler cameras for inspection pipelines.

**GitHub:** https://github.com/basler/pypylon
**Stars:** 693

## Use Cases
[UC-01 Visual Inspection](../use-cases/uc-01-visual-inspection.md)

## How It's Used
pypylon.InstantCamera provides frame capture from Basler industrial cameras. Supports hardware triggering, exposure control, and multi-camera synchronization needed for paint tunnel inspection.

## Technologies
Basler cameras, GigE Vision
