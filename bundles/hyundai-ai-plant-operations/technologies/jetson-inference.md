# jetson-inference

NVIDIA Jetson edge deployment toolkit — TensorRT optimized inference for computer vision models on Jetson Orin / Nano.

## How It's Used
Provides the runtime for deploying PyTorch/ONNX models to Jetson hardware. Handles model optimization (FP16/INT8 quantization), camera input pipelines, and inference scheduling for line-speed operation.
