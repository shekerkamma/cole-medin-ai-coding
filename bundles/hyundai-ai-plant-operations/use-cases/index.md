# Use Cases

Hyundai plant-operations use cases staged from the Obsidian vault.

* [UC-01 Visual Inspection](uc-01-visual-inspection.md) - AI-enabled cameras detect paint defects, scratches, dents, welding inconsistencies, and panel gaps in real time.
* [UC-02 Variant Confirmation](uc-02-variant-confirmation.md) - Computer vision and VIN/barcode recognition validate vehicle model, trim, and component matching during assembly.
* [UC-03 Component Validation](uc-03-component-validation.md) - AI cameras verify seat type, orientation, mounting alignment, fastening completion, and fitment accuracy.
* [UC-04 SOP Compliance](uc-04-sop-compliance.md) - Vision-based monitoring ensures operators follow defined manufacturing procedures with real-time deviation alerts.
* [UC-05 Predictive Quality](uc-05-predictive-quality.md) - ML models predict defect occurrence using production line sensor data and historical trends.
* [UC-06 Predictive Maintenance](uc-06-predictive-maintenance.md) - AI models predict machine failures and reduce unplanned downtime using vibration, acoustic, and thermal data.
* [UC-07 Safety Monitoring](uc-07-safety-monitoring.md) - Vision AI monitors PPE compliance, unsafe movements, restricted zone access, and forklift interactions.
* [UC-08 Digital Traceability](uc-08-digital-traceability.md) - End-to-end component and assembly traceability across production stages using RFID, barcode, IoT, and vision.
