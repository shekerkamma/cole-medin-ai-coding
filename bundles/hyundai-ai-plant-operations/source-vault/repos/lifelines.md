# lifelines

Survival analysis library — Weibull, Cox proportional hazards, and Kaplan-Meier models for time-to-failure estimation.

**GitHub:** https://github.com/CamDavidsonPilon/lifelines
**Stars:** 2,571

## Use Cases
[[UC-06 Predictive Maintenance]]

## How It's Used
WeibullFitter estimates failure time distributions from historical maintenance records. Cox PH model incorporates sensor covariates (vibration levels, temperature trends) to predict individual asset failure probability.

## Technologies
Weibull, Cox PH, Kaplan-Meier
