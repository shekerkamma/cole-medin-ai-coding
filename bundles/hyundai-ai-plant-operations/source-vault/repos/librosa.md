# librosa

Audio analysis library — mel spectrogram extraction, MFCC, chroma features for acoustic fault detection.

**GitHub:** https://github.com/librosa/librosa
**Stars:** 7,200

## Use Cases
[[UC-06 Predictive Maintenance]]

## How It's Used
Extracts mel spectrograms from acoustic sensor data captured near bearings, motors, and gearboxes. Spectrograms are fed into CNN classifiers to detect early bearing wear, misalignment, and lubrication degradation.

## Technologies
mel spectrogram, MFCC, chroma
